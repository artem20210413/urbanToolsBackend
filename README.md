# First launch of the project

### 1) Git clone

        - https://github.com/artem20210413/urbanToolsBackend.git

### 2) Start openserver

        - crate DB_DATABASE: "urban-tools"

### 3) Rename file .env.loc to .env

### 4) Go to the server console and navigate to the project folder.

        - composer install
        - php artisan serve
